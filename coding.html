Crafting code

Interact

This is the script for the interact mechanic. Every frame, a raycast checks to see if there is an object with a script that has a method called "Interact()". If the object does have said script, some text appears at the top confirming you are within range of an interactable object. If you press the "E" key, then you will interact with the object.
